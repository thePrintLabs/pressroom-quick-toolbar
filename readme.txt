== Description ==

Adds a 'PressRoom' dropdown to the WordPress toolbar for quick access to PressRoom functions and recently edited content Issues (published, drafts, scheduled).
This provides you quick access to PressRoom settings, developer functions and recently edited Issues (published, drafts, scheduled).

== Installation ===

1. Upload `pr-edit-toolbar.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. The 'PressRoom' dropdown menu will appear in the WordPress toolbar.
